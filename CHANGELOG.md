# Changelog

All notable changes to `@stellarroute/route-visualizer` are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).
This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial `RouteVisualizer` component with full route diagram rendering
- Per-hop slippage display with severity color-coding (green/amber/red)
- Light, dark, and auto theming via CSS custom properties
- `showSlippage`, `showPoolIds`, `maxHopsShown` configurable props
- Responsive layout: horizontal on desktop, vertical on mobile (< 480px)
- Full accessibility: ARIA roles and labels on all diagram elements
- TypeScript declarations included in package
- ESM + CJS dual-format build
- Storybook stories for all prop variants
