# Roadmap

## Current (v0.1)

- Embeddable `RouteVisualizer` React component
- AMM and orderbook hop labels
- Slippage badges with severity colors
- Light/dark/auto themes via CSS tokens
- Storybook documentation site
- npm package `@stellarroute/route-visualizer`

## Planned

| Item | Description |
|------|-------------|
| StellarRoute SDK helper | Optional utility to map quote JSON to `RouteData` |
| Animation | Subtle hop highlight on route updates |
| Accessibility audit | WCAG contrast pass on default tokens |
| Vue/Svelte wrappers | Community or official thin adapters |

## Non-goals

- Executing swaps or signing transactions
- Fetching quotes directly (host app supplies data)
- Replacing StellarRoute routing logic

Track issues at https://github.com/StellarRoute/route-visualizer/issues
