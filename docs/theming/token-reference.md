# Design token reference

All theme colors live in `src/styles/tokens.css`.

## Light / dark shared tokens

| Token | Purpose |
|-------|---------|
| `--rv-bg` | Page background |
| `--rv-surface` | Card/surface fill |
| `--rv-border` | Borders |
| `--rv-text-primary` | Primary text |
| `--rv-text-secondary` | Muted text |
| `--rv-accent` | Brand accent |
| `--rv-accent-subtle` | Accent background |
| `--rv-warning`, `--rv-warning-bg` | Warning slippage |
| `--rv-danger`, `--rv-danger-bg` | High slippage |
| `--rv-success` | Healthy/normal slippage |
| `--rv-arrow` | Connector color |
| `--rv-node-border` | Node ring |
| `--rv-font` | System font stack |
| `--rv-radius`, `--rv-radius-sm` | Corner radii |
| `--rv-shadow` | Elevated surfaces |

Dark theme overrides sit under `[data-rv-theme="dark"]`.
